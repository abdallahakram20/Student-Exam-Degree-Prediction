🎓 Student Success AI Predictor
An end-to-end Machine Learning web application that predicts student academic outcomes (Pass/Fail) based on study habits and historical performance.

🚀 Overview
This project leverages Machine Learning to provide early insights into student performance. By analyzing key metrics like daily study hours and previous exam scores, the system calculates a success probability and provides a clear "Pass" or "Fail" prediction using a trained model.

✨ Features
AI-Powered Predictions: Uses a trained ML model (Logistic Regression/Random Forest) to predict outcomes.

Dynamic UI: Modern, responsive dashboard built with Streamlit.

Real-time Analytics: Visualizes success probability with interactive Gauge charts (Plotly).

Feature Engineering: Automatically calculates student "Efficiency" metrics for higher accuracy.

Dark/Light Mode: Adaptive UI for a better user experience.

🛠️ Tech Stack
Language: Python 3.10

ML Frameworks: Scikit-Learn, Pandas, Joblib

Web Framework: Streamlit

Visualization: Plotly
